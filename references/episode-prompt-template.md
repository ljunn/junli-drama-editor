# 单集 Prompt 模板

优先用 `python3 scripts/episode_pipeline.py compose <项目目录> --episode-num <集数>` 自动生成 Prompt Pack。

如果手动拼 Prompt，至少保证下面这些上下文都在。

```text
你正在续写平台向微短剧。若出现冲突，以状态文件 > 场景计划 > 历史摘要 > 上一集剧本为准。

[粘贴 series-bible.md 全文]
[粘贴 character-design.md 全文]
[粘贴 visual-bible.md 全文]
[粘贴 narrative-style.md 全文]
[粘贴 task_log.md 全文或关键段落]

当前要写的剧集：第X集
本集标题：[标题]
本集大纲：[核心事件]
目标时长：[时长]

最近 2-3 集摘要：
[粘贴最近 2-3 集摘要]

上一集结束后的角色状态：
[粘贴 state/角色状态.md 的相关部分]

当前活跃伏笔：
[粘贴 state/伏笔列表.md 中活跃伏笔]

本集场景卡：
[粘贴 runtime/episode-XXXX.plan.md]

如果有：
- [粘贴 docs/分集梗概.md 的本集小节]
- [粘贴上一集剧本尾段]

请生成剧本，要求：
1. 总字符数≤3000
2. 默认分 4-5 个场景
3. 每个场景格式如下：
   场景X: 地点(起始秒-结束秒)
   【环境空镜Xs】描述
   (主体)角色外貌(首场景写全，后续只写服装变化)
   (环境)...
   (动作)...
   (光影)...
   (镜头)...
   (画质)...
   台词:
   【停顿Xs】
   角色名:"对话"
4. 台词不含动作和语速说明
5. 本集至少 3 个爽点
6. 不要空行，不要 --- 分隔符
7. 结尾必须留卡点
8. 不能让角色越权说出不该知道的信息
```
