# 平台投稿资料包

无论投稿哪个平台，短剧项目至少准备下面 6 项。

## 1. 短剧概况

包含：

- 剧名
- 类型
- 题材方向
- 总集数
- 单集时长
- 受众
- 付费卡点或看点
- 300 字以内故事梗概

## 2. 人物小传

核心角色建议 2-4 人，每人 100-300 字，至少包含：

- 基础信息
- 外观造型
- 性格标签
- 表面身份 / 隐藏身份
- 核心技能或金手指
- 情感线
- 剧情主线

## 3. 故事大纲

至少覆盖：

- 背景
- 切入点
- 铺垫
- 高潮
- 结尾与后续钩子

## 4. 分集梗概

每集 100-300 字，按“起因 - 经过 - 结果 / 卡点”结构写。

## 5. 单集剧本

至少准备样集和最新完成集。默认使用本仓库的结构化场景块格式。

## 6. 状态说明

建议额外附上：

- 角色知情状态
- 当前活跃伏笔
- 已完成集数
- 后续主线方向

这样平台、导演或合作编剧更容易接手。
