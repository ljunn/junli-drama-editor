# 单集质量检查清单

交付前逐项核对。先跑命令做结构门，再做人审。

## 命令可自动检查

- [ ] 有 `场景X:` 场景块
- [ ] 场景标题带合法时长范围
- [ ] 每个场景包含环境空镜、主体、动作、光影、镜头、画质、台词
- [ ] 总字符数不超过 3000
- [ ] 正文没有短到只剩提纲感
- [ ] 没有空行
- [ ] 没有 `---` 分隔符
- [ ] 风险词已巡检
- [ ] 台词行里没有动作或语速说明
- [ ] `台词:` 后存在有效对白行

## 人工必查

- [ ] 本集是剧本，不是小说段落
- [ ] 默认 4-5 个场景
- [ ] 独立地点切换不超过 3 次
- [ ] 结尾有卡点
- [ ] 3 秒内进入冲突
- [ ] 至少 3 个爽点
- [ ] 至少 1 个反转或信息升级
- [ ] 每场至少完成“目标 -> 阻碍 -> 变化”三步
- [ ] 对白和动作足够密，不是每场只讲一个信息点
- [ ] 台词短、狠、有信息量
- [ ] 没有无效对话和水词

## 人物人工复核

- [ ] 首场景主角外貌完整
- [ ] 后续场景只写服装或造型变化
- [ ] 角色没有说出自己不该知道的秘密
- [ ] 人物行为符合既有设定
- [ ] 角色没有 OOC

## 风险词巡检

命令会提示这些词，但要人工判断是否真在偷写小说：

- `心想`
- `内心`
- `觉得`
- `感觉`
- `紧张`
- `难过`
- `伤心`
- `愤怒`
- `仿佛`
- `似乎`
- `空气里`
- `氛围`
- `意识到`

如果出现，优先改成动作、表情、道具、造型、灯光或镜头。

## 字数

- [ ] 超标时先删空行、分隔符、重复人物描述
- [ ] 再精简镜头和光影形容词
