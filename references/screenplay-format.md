# 微短剧剧本格式规范

## 核心原则

1. 剧本是视觉化拍摄文本，不是小说。
2. 所有文字都必须可拍、可演、可拆镜。
3. 微短剧优先短句、快节奏、强信息差。

## 默认输出格式

```text
场景1: 订婚宴后台(0-45s)
【环境空镜5s】化妆间，镜前灯昏黄
(主体)苏若微（23岁东方女性，黑色长卷发披肩，瓜子脸，桃花眼，红唇，白色高定礼服），林子枫（蓝色西装），白芊芊（白裙）
(环境)狭窄后台，梳妆镜，散落的化妆品
(动作)苏若微猛然睁眼。林子枫搂着白芊芊亲吻。苏若微按下手机录音键。
(光影)镜前灯昏黄高光，其余沉入阴影
(镜头)5s眼部特写，切二人中景
(画质)豪门质感
台词:
【2s急促呼吸】
林子枫:"订婚结束，我就和她分手。"
```

## 场景块要求

每个场景至少包含：

- 场景标题和时长范围
- `【环境空镜Xs】`
- `(主体)`
- `(动作)`
- `(光影)`
- `(镜头)`
- `(画质)`
- `台词:`

可选增加：

- `(环境)`
- `场景转换:`

## 首场景与后续场景

- 首场景：主角完整外貌必须写清楚
- 后续场景：只写服装变化或关键造型变化

## 台词规则

1. 台词单独成行。
2. 不同角色台词分开。
3. 台词不夹动作、语速或镜头说明。
4. 每句台词都必须有功能：
   - 推剧情
   - 塑人物
   - 交代信息
   - 递进关系

## VO / OS

- `角色名（VO）:` 人在画面中，但出现短独白
- `角色名（OS）:` 人不在画面中，只有声音

慎用原则：

- VO 只在动作无法传达时补最低限度心理信息
- OS 常用于电话、门外说话、广播、旁白

## 特殊场景

### 闪回 / 梦境

```text
闪回开始
场景3: 旧宅走廊(90-115s)
...
闪回结束
```

### 对切

```text
场景4: 女主卧室/陆家书房(135-180s)
...
```

### 空镜转场

```text
场景转换: 45s处 Hard cut
```

## 常见错误

### 错误 1：心理描写

```text
苏若微心里又恨又怕。
```

改成：

```text
苏若微后槽牙咬紧，指甲掐进掌心。
```

### 错误 2：环境抒情

```text
空气里弥漫着令人窒息的绝望。
```

改成：

```text
走廊顶灯忽明忽暗，尽头空无一人。
```

### 错误 3：台词没用

```text
林子枫:"嗯……那个……你先听我解释。"
```

改成：

```text
林子枫:"录音是断章取义。"
```

## 4-5 场景与 3 次场景转换的处理原则

这两个要求不是硬冲突，按下面执行：

- AI 视频生成版：默认 4-5 个场景块，方便拆镜和生成
- 拍摄版：独立地点切换不超过 3 次
- 如果同一地点只是在节奏上分成两段，可以保留两个场景块，但地点命名保持一致
