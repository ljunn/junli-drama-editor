# 微短剧剧本格式规范

## 核心原则

1. 剧本是视觉化拍摄文本，不是小说。
2. 所有文字都必须可拍、可演、可拆镜。
3. 微短剧优先短句、快节奏、强信息差。

## 默认输出格式

先分清当前交付层级，再套格式：

- 整集 / 分场正文：用“场景块格式”
- 5 秒视频拆镜：用“单镜头格式”
- 不要把两套格式混写

## 1. 场景块格式

适用：

- 整集剧本
- 按场景交付的 AI 视频版
- 拍摄版的单场正文

```text
场景1: 订婚宴后台(0-45s)
【环境空镜5s】化妆间，镜前灯昏黄
(主体)苏若微（23岁东方女性，黑色长卷发披肩，瓜子脸，桃花眼，红唇，白色高定礼服），林子枫（蓝色西装），白芊芊（白裙）
(环境)狭窄后台，梳妆镜，散落的化妆品
(动作)苏若微猛然睁眼。林子枫搂着白芊芊亲吻。苏若微按下手机录音键。
(光影)镜前灯昏黄高光，其余沉入阴影
(镜头)先给苏若微眼部特写，再切林子枫和白芊芊的中景对切
(画质)豪门质感
台词:
【2s急促呼吸】
林子枫:"订婚结束，我就和她分手。"
```

说明：

- 场景块可以覆盖 20 秒、30 秒、45 秒等一整场时长
- `(镜头)` 写的是本场的镜头设计，不是“每 5 秒一个镜头”的拆分表
- 如果当前任务要求严格 5 秒一镜，禁止继续使用这套格式输出最终结果

## 2. 5秒单镜头格式

适用：

- `compose-shots`
- 下游工具一次只能吃 5 秒镜头
- 用户明确要求“每 5 秒一个镜头”

```text
镜头1: 订婚宴后台(0-5秒)
(主体)苏若微（23岁东方女性，黑色长卷发披肩，瓜子脸，桃花眼，红唇，白色高定礼服）站在镜前，肩颈骤然绷紧。
(环境)昏黄镜前灯压住狭窄化妆间，梳妆镜边缘散着化妆刷和粉盒。
(动作)苏若微猛然睁眼，视线从镜中锁向后方重叠的人影，右手立刻滑向手机侧键。
(光影)镜前暖黄高光打亮她的瞳孔，背景人影压在半暗里。
(镜头)固定近景起，0-3秒锁苏若微眼神变化，3-5秒轻推到她按下录音键的手部。
(画质)豪门质感，人物边缘清晰，镜面高光可控。
台词:
【急促呼吸】
```

硬约束：

- 一个文件只写一个镜头
- 秒数必须是严格连续 5 秒，如 `0-5秒`、`5-10秒`
- 一个镜头只放 1 个主要动作节拍
- 台词默认 0-1 句；如果需要第 2 句，拆到下一个镜头
- 不要再写 `场景1: 地点(0-45s)` 这种整场标题

## 场景块要求

每个场景块至少包含：

- 场景标题和时长范围
- `【环境空镜Xs】`
- `(主体)`
- `(动作)`
- `(光影)`
- `(镜头)`
- `(画质)`
- `台词:`

可选增加：

- `(环境)`
- `场景转换:`

每个 5 秒单镜头至少包含：

- 镜头标题和时长范围
- `(主体)`
- `(动作)`
- `(光影)`
- `(镜头)`
- `(画质)`
- `台词:`

单镜头可选增加：

- `(环境)`

## 首场景与后续场景

- 首场景：主角完整外貌必须写清楚
- 后续场景：只写服装变化或关键造型变化
- 如果是单镜头模式，只有本集首镜头需要写主角完整外貌；同场后续镜头只写服装或关键变化

## 台词规则

1. 台词单独成行。
2. 不同角色台词分开。
3. 台词不夹动作、语速或镜头说明。
4. 每句台词都必须有功能：
   - 推剧情
   - 塑人物
   - 交代信息
   - 递进关系

## VO / OS

- `角色名（VO）:` 人在画面中，但出现短独白
- `角色名（OS）:` 人不在画面中，只有声音

慎用原则：

- VO 只在动作无法传达时补最低限度心理信息
- OS 常用于电话、门外说话、广播、旁白

## 特殊场景

### 闪回 / 梦境

```text
闪回开始
场景3: 旧宅走廊(90-115s)
...
闪回结束
```

### 对切

```text
场景4: 女主卧室/陆家书房(135-180s)
...
```

5 秒单镜头模式下，对切也仍然只能覆盖当前 5 秒，不能借“对切”把两个动作节拍塞回同一镜头。

### 空镜转场

```text
场景转换: 45s处 Hard cut
```

如果是 5 秒单镜头模式，转场应该体现在镜头承接里，而不是重新退回整场格式。

## 常见错误

### 错误 1：心理描写

```text
苏若微心里又恨又怕。
```

改成：

```text
苏若微后槽牙咬紧，指甲掐进掌心。
```

### 错误 2：环境抒情

```text
空气里弥漫着令人窒息的绝望。
```

改成：

```text
走廊顶灯忽明忽暗，尽头空无一人。
```

### 错误 3：台词没用

```text
林子枫:"嗯……那个……你先听我解释。"
```

改成：

```text
林子枫:"录音是断章取义。"
```

### 错误 4：要求 5 秒一镜，却还在写整场

```text
场景1: 订婚宴后台(0-45s)
【环境空镜5s】...
(主体)...
```

改成：

```text
镜头1: 订婚宴后台(0-5秒)
(主体)...
(动作)...
```

## 4-5 场景与 3 次场景转换的处理原则

这两个要求不是硬冲突，按下面执行：

- AI 视频生成版：默认 4-5 个场景块，方便拆镜和生成
- 拍摄版：独立地点切换不超过 3 次
- 如果同一地点只是在节奏上分成两段，可以保留两个场景块，但地点命名保持一致
- 如果进入严格 5 秒工作流，先按场景规划，再拆成单镜头；不要拿单镜头输出反向替代场景卡
